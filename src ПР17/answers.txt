1. Управляемые компоненты управляются состоянием React, а неуправляемые — напрямую DOM.
2. Управляемые используют при частом изменении и валидации, неуправляемые — для простых форм или интеграции с кодом, не использующим React.
3. onChange для ввода, onSubmit для отправки, onClick для кликов.
4. Использовать один обработчик, определяя поле по атрибуту name или другому идентификатору.
5. Встроенная HTML-валидация, кастомная валидация в обработчиках, сторонние библиотеки (Formik, React Hook Form).
6. Вызвать event.preventDefault() в обработчике события onSubmit.
7. Передача состояния формы вверх по иерархии компонентов для общего доступа или управления.
