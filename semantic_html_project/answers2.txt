Ответы на вопросы
1. Семантическая верстка использует теги по смыслу, а не для внешнего вида. Важна для доступности и SEO.

2. <article> для независимого контента (статья о пошиве), <section> для логических разделов внутри.

3. <header> и <footer> для шапки и подвала. Да, могут использоваться多次но в разных секциях.

4. <main> для основного контента. <div> не несет смысловой нагрузки.

5. <nav> для основной навигации. Обычный список - просто группировка ссылок. <nav> обязательно для главного меню.

6. datetime машинно-читаемый формат времени. Важен для поисковиков и accessibility.

7. <figure> и <figcaption> семантически связывают изображение с подписью. Улучшают доступность.

8. <aside> для дополнительного контента (сайдбар с услугами). Не основная информация.

9. Улучшает индексацию поисковиками и помогает скринридерам для слабовидящих.

10. <header>, <nav>, <main>, <article>, <section>, <aside>, <footer> - помогают ориентироваться в структуре страницы.
