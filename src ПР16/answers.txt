1. useState - для простого состояния, useReducer - для сложной логики состояний

2. Массив зависимостей определяет, когда запускать эффект. Без него - при каждом рендере

3. Stale closure - устаревшие значения. Используйте функциональные обновления setState(prev => prev + 1)

4. useCallback/useMemo нужны при передаче функций/объектов в дочерние компоненты, не для преждевременной оптимизации

5. useRef - для DOM-элементов и изменяемых значений без перерендера, useState - для состояния с перерендером

6. Правила хуков: вызывать только на верхнем уровне и в React-функциях

7. Кастомный хук - функция, начинающаяся с use, использующая другие хуки

8. useContext упрощает передачу данных через компоненты без prop drilling

9. Асинхронные операции - через useEffect с cleanup функцией

10. Отладка хуков: React DevTools, console.log, правила хуков
