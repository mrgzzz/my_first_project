1. Viewport meta tag управляет масштабированием и размерами области просмотра на мобильных устройствах. 

2. Использовала Responsive (отзывчивый) подход с медиа-запросами и Fluid (гибкий) с процентными размерами.

3. Mobile First - стили пишутся сначала для мобильных, затем добавляются для больших экранов. Преимущества: лучшая производительность, прогрессивное улучшение.

4. % для контейнеров, rem для типографики, vw/vh для полноэкранных элементов. Они относительные и масштабируются.

5. clamp(min, preferred, max) задает гибкий размер шрифта, который изменяется между min и max значениями в зависимости от размера экрана.

6. 480px (мобильные), 768px (планшеты), 1024px (десктоп). Определил по распространенным разрешениям устройств.

7. @media (max-width: 768px) { стили } - применяет стили при ширине экрана до 768px.

8. max-width - стили применяются ДО указанной ширины, min-width - ОТ указанной ширины. max-width для Desktop First, min-width для Mobile First.

9. img { max-width: 100%; height: auto; } - изображение не превышает ширину контейнера и сохраняет пропорции.

10. Инструменты разработчика: Responsive Design Mode, эмуляция устройств, изменение размеров окна, панель Computed Styles.
