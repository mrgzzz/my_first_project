1. Function Declaration поднимается, Function Expression - нет
2. Rest parameters - массив, arguments - объект
3. Замыкание - функция + её лексическое окружение
4. Рекурсия для древовидных структур, итерация для циклов
5. Функции принимающие/возвращающие функции: map, filter, reduce
6. Каррирование - преобразование функции для последовательного вызова
7. Мемоизация кэширует результаты для оптимизации
8. Debounce - откладывает, throttle - ограничивает частоту
9. Генераторы можно приостанавливать (yield)
10. Применены: чистые функции, композиция, функции высшего порядка
